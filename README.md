# Sistema de Leilões - TDSat

## Descrição do Projeto
Sistema desenvolvido para gerenciar as rotinas de uma casa de leilões, permitindo o cadastro e acompanhamento de itens. Este projeto faz parte da Unidade Curricular de Gerenciar a configuração e versionamento de Software.

## Tecnologias Utilizadas
* Java (NetBeans)
* MySQL (Banco de Dados) 
* Git & GitHub (Versionamento)
